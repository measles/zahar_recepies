# Рэцэпты Андрэя і Алены Захарэвічаў #

Рэцэпты захоўваюцца ў фармаце markdown як для [Kookbook](https://github.com/KDE/kookbook). Але каб скарыстацца будзе дастаткова захаваць рэцэпты на дыск і проста адкрыць іх якім заўгодна тэкставым рэдактарам.

Усе рэцэпты хаця б аднойчы былі прыгатаваныя намі і ўсе мы палічылі смачнымі. Частку з іх мы крыху перапрацавалі пад уласны густ ці наогул прыдумалі самі.

Смачна есці!
